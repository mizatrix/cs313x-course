About the Porter Stemmer

Published by Martin Porter in 1980, this is the most widely used English stemmer. It works in 5 steps, each applying a set of rules based on:

  • Measure m — the number of vowel-consonant sequences ([C](VC){m}[V]). Determines if a stem is "long enough" to strip further.
  • Conditions — like *v* (stem contains a vowel), *d (ends in double consonant), *o (ends in CVC where last ≠ w/x/y).
  • Suffix rules — each step targets specific suffix categories: plurals → tense → derivational → adjective → final cleanup.

📜 Algorithm Pseudocode

PORTER-STEM(word)
 // Step 1a: Remove plurals
 1  SSES → SS  |  IES → I  |  SS → SS  |  S → ∅

 // Step 1b: Remove past tense / progressive
 2  if (m > 0) EED → EE
 3  if (*v*) ED → ∅     // stem must contain vowel
 4  if (*v*) ING → ∅
 5    then fix: AT→ATE, BL→BLE, IZ→IZE
 6           or  remove double (except LL, SS, ZZ)
 7           or  (m=1 and *o) → add E

 // Step 1c: Y → I
 8  if (*v*) Y → I

 // Step 2: Map double suffixes (m > 0)
 9  ATIONAL → ATE  |  TIONAL → TION  |  ENCI → ENCE
10  ANCI → ANCE    |  IZER → IZE     |  ALISM → AL  ...

 // Step 3: Derivational morphology (m > 0)
11  ICATE → IC  |  ATIVE → ∅  |  ALIZE → AL  ...

 // Step 4: Remove suffixes (m > 1)
12  AL, ANCE, ENCE, ER, IC, ... → ∅
13  ION → ∅  only if stem ends in S or T

 // Step 5: Final cleanup
14  if (m > 1) E → ∅
15  if (m = 1 and not *o) E → ∅
16  if (m > 1 and *d and *L) remove last letter

Notation:  m = measure [C](VC){m}[V]
  *v* = stem contains vowel  *d = ends double consonant
  *o = ends CVC (where last ≠ W, X, Y)
