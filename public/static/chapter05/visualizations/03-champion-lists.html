Explore how search engines use heuristics to avoid scoring every document. Compare full scoring vs. champion lists vs. index elimination.

Key Trade-off: These heuristics sacrifice recall (might miss some relevant docs) for massive speed gains. Is the trade-off worth it?

📊 Document Scores
📖 How It Works
Full Scoring: ✅ Exact results, ❌ Slow (O(N)) — use for small collections.
Champion Lists: ✅ Pre-computed top-r per term, ❌ May miss some docs — best general approach.
Index Elimination: ✅ Fast by skipping low-IDF terms, ❌ May miss docs — use when query has many terms.
